# 🔬 Raman ML: Классификация экспрессии белка HSP70 по рамановским спектрам

Данный репозиторий содержит робастный Machine Learning пайплайн для бесконтактной детекции стресс-ответа биологических тканей (экспрессии шаперона HSP70) in vitro на основе данных рамановского картирования (Raman Mapping).

Решение разработано в рамках хакатона и обеспечивает точность классификации до **92.86%** при строгой Patient-Level валидации.

## Описание задачи
Анализ рамановских спектрограмм тканей мозга (Стриатум, Кора, Мозжечок) для классификации образцов на 3 группы:
- `control` — контрольная группа (здоровая ткань);
- `endo_hsp70` — эндогенная экспрессия белка HSP70;
- `exo_hsp70` — экзогенная экспрессия белка HSP70.

**Дополнительная задача:** Извлечение биологически интерпретируемых признаков (Feature Importance) для локализации спектральных маркеров болезни.

## Архитектура решения

Мы отказались от "слепого" применения нейросетей (CNN) и снижения размерности (PCA) в пользу физически обоснованного конвейера и ансамбля деревьев. 

### 1. Физический Препроцессинг Спектров
- **Despiking (MedFilt):** Удаление аппаратных артефактов ("космических лучей") от CCD-матриц спектрометра.
- **ALS (Asymmetric Least Squares):** Алгоритмическое удаление флуоресцентного фона (baseline correction) без искажения формы рамановских пиков.
- **Savitzky-Golay:** Консервативное сглаживание шума.
- **SNV Normalization:** Стандартная нормализация для компенсации разницы в толщине гистологических срезов.

### 2. ML-Движок (Двухуровневый инференс)
- **Уровень пикселя:** Параллельное предсказание класса каждым спектром (вокселем 1x1 мкм) с помощью Soft Voting ансамбля: **XGBoost + Random Forest**.
- **Уровень пациента (Majority Voting):** Агрегация сотен предсказаний из одного сканирования в финальный диагноз для конкретного животного, что поглощает статистические выбросы.
- **Валидация:** Строгий *GroupKFold* по ID животных (Data Leakage Protection). Модель тестируется только на "незнакомых" организмах.

## Результаты

Лучшие метрики получены на диапазоне **1500 см⁻¹ (Fingerprint region)**, где расположены амидные связи белков:

| Регион Мозга | Диапазон | Точность (Patient-Level CV) |
| :--- | :--- | :---: |
| **Cerebellum (Мозжечок)** | 1500 cm⁻¹ | **92.86%**  |
| **Cortex (Кора)** | 1500 cm⁻¹ | **79.63%**  |
| **Striatum (Стриатум)** | 1500 cm⁻¹ | **75.00%**  |

*Примечание:* Алгоритм успешно подтвердил отсутствие биологического сигнала в диапазоне 2900 см⁻¹ для коры мозга (падение метрики до уровня случайного угадывания), что доказывает отсутствие переобучения на аппаратный шум.

## Поиск Биомаркеров (Интерпретируемость)
В пайплайн встроен модуль экстракции `feature_importances_`. Модель автоматически подсвечивает для биоинформатиков те волновые числа (см⁻¹), которые вносят наибольший математический вклад в разделение классов.
